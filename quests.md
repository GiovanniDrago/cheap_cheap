# Quests

## Add a profile picture
- Description: Set a profile image in the profile section.
- Repeatable: No

## Add an expense
- Description: Add a single expense entry.
- Repeatable: Daily

## Add 3 expenses
- Description: Add three expenses in a single day.
- Repeatable: Daily

- Description: Create a new category from the category list or the expense form.
- Repeatable: Daily

## Open category list
- Description: Open the categories section.
- Repeatable: No

## Add a recurrent expense
- Description: Save an expense with recurrence enabled.
- Repeatable: Weekly
